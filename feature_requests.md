# Feature Requests

## Planned Features

### Show Cursor
- Display cursor position in terminal output
- Visual indicator for where text will be inserted

### Voice Mode
- Voice input for sending commands to agents
- Speech-to-text for hands-free interaction
- Possibly voice output for agent responses

### Claude Hooks
- Integration with Claude Code hooks system
- Custom actions triggered by agent events
- Automated responses to specific prompts

### Notifications
- Push notifications when agents need input
- Alert when pane status changes to waiting
- Background notifications when app is not in focus

### Proper PWA Install
- HTTPS support for full PWA functionality
- Automatic install prompt on supported browsers
- Offline support with service worker caching

### Responsive Rendering on Mobile
- Improved terminal output rendering for small screens
- Better handling of wide content / horizontal scrolling
- Adaptive font sizes based on screen width
